# Claude Ultimate Guide 2026

Интерактивный гайд по работе с Claude от [@dmitriymarketing](https://t.me/dmitriymarketing).

## Структура

```
claude-guide-deploy/
├── index.html          ← главная страница
├── assets/             ← все скриншоты и обложки
│   ├── image.png
│   ├── image-1.png … image-13.png
│   ├── cover-5.jpg     ← обложка YouTube «Лид-магнит за 2 минуты»
│   ├── cover-7.jpg     ← обложка YouTube «ИИ-агент в Telegram»
│   └── claude-code.jpg ← скриншот Claude Code (см. ниже)
├── vercel.json
└── README.md
```

## Деплой на Vercel

1. Зайди на [vercel.com](https://vercel.com), залогинься через GitHub
2. **Add New → Project** → импортируй репозиторий с этой папкой
3. Framework preset → **Other** (это статический HTML)
4. Build Command — оставь пустым
5. Output Directory — оставь пустым (или `.`)
6. Жми **Deploy**

Vercel сам подхватит `index.html` из корня и через 30 секунд выдаст домен типа `claude-guide-xxx.vercel.app`. Можно подключить свой домен в Settings → Domains.

## Деплой на GitHub Pages

1. Создай репозиторий на GitHub, залей содержимое этой папки в корень
2. Settings → Pages → Source → **main branch / root**
3. Сохрани, через минуту страница доступна по адресу `username.github.io/repo-name`

## Деплой через простой хостинг (Beget, REG.ru и т.п.)

1. Через FTP/файловый менеджер закинь все файлы в папку сайта (обычно `/public_html/` или `/www/`)
2. Открой домен — гайд будет доступен сразу

## Один файл нужно подложить вручную

В папке `assets/` пока **нет** одного файла:
- **`claude-code.jpg`** — скриншот Claude Code из Notion-страницы (раздел Блок 6 → Claude Code)

Открой свою Notion-страницу → правый клик на скриншоте Claude Code → **«Сохранить картинку как…»** → сохрани в `assets/claude-code.jpg`. После этого скриншот появится на странице.

Если не подложишь — на странице будет дыра в этом месте (картинка просто не отобразится, ничего не сломается).

## Обновление контента

В деплой включена встроенная **Edit-mode панель** (правый нижний угол на странице → кнопка «Редактировать», или горячая клавиша `Ctrl+E` / `Cmd+E`). Через неё ты можешь:

- Редактировать любой текст прямо в браузере
- Менять акцентные цвета через color-picker
- Применять готовые палитры
- Скачать обновлённый HTML кнопкой «⬇️ Скачать готовый HTML»

Эта панель видна только тебе (или тому кто откроет страницу) и в обычном виде свернута. Конечному пользователю она не мешает — просто маленькая кнопка в углу.

## Тех. детали

- Чистый HTML/CSS/JS, без сборки и билда
- Работает на любом статическом хостинге
- Шрифты Gilroy (через CDNFonts) + Montserrat fallback (через Google Fonts) — подгружаются при открытии страницы
- Темы: светлая и тёмная (тогл-кнопка в сайдбаре, выбор сохраняется в localStorage)
- Адаптивная вёрстка: десктоп / планшет / мобилка
